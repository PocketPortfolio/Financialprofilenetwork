# Demo GIF Script - Local-First CSV Import

## Overview

Create a 10-15 second GIF demonstrating the local-first CSV import feature for Reddit/Hacker News launch.

## Script

### Scene 1: Landing Page (2 seconds)
- Show clean landing page
- Highlight "Upload CSV" area
- Text overlay: "Privacy-First Portfolio Tracker"

### Scene 2: CSV Drag & Drop (2 seconds)
- User drags CSV file over upload area
- Visual feedback (highlighted border)
- Text overlay: "Drag & Drop Your Broker CSV"

### Scene 3: Instant Parsing (2 seconds)
- File is selected
- Show "Parsing..." indicator briefly
- Text overlay: "Auto-Detecting Broker..."

### Scene 4: Broker Detection (2 seconds)
- Show "Detected: Robinhood" badge
- Show parsed trade count
- Text overlay: "15+ Brokers Supported"

### Scene 5: Dashboard Render (3 seconds)
- Dashboard appears with portfolio data
- Show pie chart, positions table
- Text overlay: "No Signup Required - 100% Local"

### Scene 6: Final Frame (2 seconds)
- Show full dashboard
- Text overlay: "Try it: pocketportfolio.app"
- GitHub star CTA

## Technical Requirements

### Tools
- **Windows:** ScreenToGif, OBS Studio
- **Mac:** Kap, QuickTime
- **Linux:** Peek, SimpleScreenRecorder

### Settings
- **Resolution:** 1280x720 (720p)
- **Frame Rate:** 30 FPS
- **Duration:** 10-15 seconds
- **File Size:** < 5MB (optimize with GIF compression)
- **Format:** GIF or MP4 (GIF for Reddit, MP4 for HN)

### Optimization
- Use color palette reduction
- Limit to 256 colors
- Reduce frame rate if needed (15-20 FPS acceptable)
- Compress with tools like:
  - EZGIF.com
  - ImageOptim
  - Gifsicle

## Recording Tips

1. **Prepare Environment:**
   - Clear browser cache
   - Use sample CSV file ready
   - Close unnecessary tabs
   - Use clean browser theme

2. **Recording:**
   - Record at 2x speed, slow down in post
   - Use smooth mouse movements
   - Pause between actions for clarity
   - Add text overlays in post-production

3. **Post-Production:**
   - Add text overlays
   - Highlight important elements
   - Add smooth transitions
   - Optimize file size

## Sample CSV for Demo

Use a small, clean CSV file:
- 5-10 trades
- Clear broker format (Robinhood recommended)
- Recent dates
- Popular stocks (AAPL, MSFT, GOOGL)
