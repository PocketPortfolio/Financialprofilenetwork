**Sovereign Engineering · Part 5**

## Guest: `localPortfolioStore.ts`

```typescript
const STORAGE_KEY_TRADES = 'pocket-portfolio-local-trades';
```

Trades JSON + metadata; quota estimated vs **5MB** budget — fail closed when over threshold.

## Auth: `useTrades` → `TradeService.getTrades(uid, forceServerFetch)`

Cloud is authoritative for signed-in users; UI still filters **healed** IDs from `localStorage`.

## Prefs: `portfolioStore` `persist` + `partialize`

Only chart view, time range, sectors, benchmarks — **not** full positions blob.

## Logout: `useAuth` → `clearIndexedDbPersistence` + `terminate(db)`

Clears **Firestore client cache** (IndexedDB), not “the vault” for guests.
