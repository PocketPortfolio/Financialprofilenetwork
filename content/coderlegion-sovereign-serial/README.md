## How to use

1. Open the `.md` file for the post you want to publish.
2. Copy the **title**, **date**, and **tags** from the first line (HTML comment) into CoderLegion’s post form.
3. Copy the **body** (from the `# Title` line through the final CTA) into the post body.
4. If CoderLegion supports a cover image, use the chapter header URL, e.g. `https://www.pocketportfolio.app/book-assets/assets/chapter-headers/chapter-01-header.svg` for post 1.

Canonical (source of truth) posts live on the site as MDX in `content/posts/sovereign-serial-*.mdx`. Set canonical URL on CoderLegion to `https://www.pocketportfolio.app/blog/{slug}` when available.
