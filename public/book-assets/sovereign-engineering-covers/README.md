# Sovereign Engineering — Amber Terminal covers

Twelve **PNG** covers for the CoderLegion **Sovereign Engineering** serial (`se-01.png` … `se-12.png`). Raster is **2400×1260** (2× of 1200×630) for crisp Open Graph and social previews. Pocket Portfolio serves **PNG/JPEG** for cover reliability; SVG is not used for these assets.

- **Regenerate:** from repo root, `node scripts/generate-sovereign-engineering-cover-pngs.mjs` (uses `@resvg/resvg-js`).
- **Palette:** near-black surfaces, accent **`#f59e0b`** (brand amber); Part 8 email mock uses aligned dark chrome.
- **Typography:** system UI stack (Segoe UI / Helvetica Neue) — clear, product-grade, no stock photography.
- **Usage in YAML:** `cover_image: "https://www.pocketportfolio.app/book-assets/sovereign-engineering-covers/se-NN.png"`
- **Usage in MDX:** `image: /book-assets/sovereign-engineering-covers/se-NN.png`
